The Slow Laptop

So one day I got a laptop for free. It was an older Toshiba Satellite with an AMD A6 chip and 6GB RAM. It ran Windows 8.1, also known as one of the worst operating systems of all time. And it was really, really slow and loud. So I had an idea. I was going to turn it into a Google laptop. I was going to turn it into a..... chromebook.

So I went on a new computer and looked up, "How can I hack a computer to be a chromebook" and I learned about ChromeOS Flex. It's a version of ChromeOS that can be installed on any computer. So I downloaded the Chrome extension "Chromebook Recovery Tool" and it said "insert a USB." I did, I selected ChromeOS Flex, and it made a bootable USB for me.

I put the USB into the old laptop and turned it on. It booted to Windows 8 still. Then I remembered: I have to change the boot order in the BIOS! So I went into the BIOS and changed the order, booted it, and it said, "installing chromeos flex" with a big Chrome logo. Once it was installed, it rebooted and said "log in." It was even slower than it was running Windows 8! I was a bit mad. But the end results were still better. I had a great lightweight computer for browsing the web, sending emails, and typing things on. It was slow, slow at doing literally every single computer task possible, but at least I don't have to use IE 8 with the worst OS ever. The End!
